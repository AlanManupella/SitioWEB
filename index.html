1.Diferencia entre GIT y GITHUB

GIT-es una herramienta que puedes utilizar en tu computadora para guardar cambios y versiones de tu codigo,es util para llevar un historial de lo que hiciste, Como un "control de versiones" que funcionan Localmente en tu PC

En el caso de GITHUB-Es una pagina web donde puedes subir proyectos que utilizan Git, para guardarlos en la nube y compartirlos en internet, es como una nube para tus proyectos
